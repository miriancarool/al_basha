# Site institucional one page

Desenvolvido para um restaurante árabe, com foco em apresentação visual e experiência do usuário.

Animações acionadas por scroll utilizando IntersectionObserver, requestAnimationFrame, scrollY
Seção “Sobre nós” com apresentação do restaurante
Integração com mapa via Google Maps (embed)
Botão de contato
Rodapé com animação contínua

Tecnologias usadas: HTML5, CSS3 e JavaScript.
